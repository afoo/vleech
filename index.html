Vleech

Vleech is a simple video downloader for sites like youtube. It currently supports the following sites:

  • youtube.com
  • break.com
  • video.google.com
  • myvideo.de
  • guba.com

I decided to create yet another of those tools because I wanted one with a really simple plugin system so support for new sites can be added easily.

The latest release of vleech is v0.2. You can fetch it from here.

You can follow the development on github. Patches (especially for new site plugins) are always welcome.
